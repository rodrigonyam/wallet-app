# Wallet Application

This is a simple wallet application that allows users to take pictures of their credit cards and save them for easy access instead of carrying the physical cards.

## Project Structure

```
wallet-app
├── index.html        # Main HTML document for the wallet application
├── css
│   └── styles.css    # Styles for the wallet application
├── js
│   ├── app.js        # Main JavaScript file for user interactions
│   ├── camera.js     # Functions for accessing the device camera
│   └── storage.js    # Manages storage of captured images
└── README.md         # Documentation for the project
```

## Features

- Capture images of credit cards using the device camera.
- Save captured images for easy access.
- View saved credit card images within the application.
- Delete saved images when no longer needed.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open `index.html` in your web browser to run the application.
3. Ensure that your browser has permissions to access the camera for capturing images.

## Usage Guidelines

- Click on the capture button to take a picture of your credit card.
- After capturing, you can save the image for future access.
- Navigate through saved images and delete any that are no longer needed.
